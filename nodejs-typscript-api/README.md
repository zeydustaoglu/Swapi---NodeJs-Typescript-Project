# NodeJs/Typescript Starwars Api Project
Version 0.1.0

## The Api
Start Wars API: https:swapi.dev/

## Migrations
To manage changes in database schema
```
prisma migrate dev
```

## Using command
To retrieve resources from the API and create them in the database
```
tsc seed.ts
```
```
node seed.js import
```
## Running Prisma Studio
```
npx prisma studio
```

## Running The App
```
npm run build
```
```
npm start
```
